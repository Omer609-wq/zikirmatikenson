# Çok dil (i18n) yapısı

## Dosyalar

| Yol | İçerik |
|-----|--------|
| `locales/tr.json` | Arayüz metinleri (Türkçe, kaynak dil) |
| `locales/id.json` | Endonezyaca arayüz |
| `locales/ms.json` | Malayca arayüz |
| `locales/en.json` | İngilizce arayüz |
| `data/library/tr.json` | Keşfet kütüphanesi (dua/zikir) |
| `data/library/premium-tr.json` | Premium ek kütüphane |
| `data/library/id.json` | Endonezyaca çeviriler (aynı `id`, sadece çevrilen alanlar) |
| `data/library/premium-id.json` | Premium Endonezyaca overlay |
| `i18n.js` | `t()`, locale, kütüphane birleştirme |

## Yeni arayüz metni

1. `locales/tr.json` içine anahtar ekle (ör. `settings.foo`).
2. `index.html` veya JS içinde `data-i18n="settings.foo"` veya `t('settings.foo')`.
3. Aynı anahtarı `en.json`, `id.json`, `ms.json` dosyalarına çevir.

## Yeni kütüphane maddesi

1. `data/library/tr.json` sonuna nesne ekle (`id`, `category`, `name`, `meaning`, `context`, `source`, `target`, `keywords`).
2. Çeviri için `data/library/id.json` içine **aynı `id`** ile sadece çevrilen alanları yaz.
3. `app.js` içine dizi **ekleme** — gerekirse `node scripts/extract-library-to-json.cjs` ile `app.js`'den export (geçici).

## Editöryal kurallar (kütüphane)

- `meaning`: doğru, sade meal
- `context`: muteber özet; kesin vaat iddiasından kaçın
- `source`: mümkünse kitap/bab
- `target`: rivayete uygun; değilse düşük tut ve `context`'te açıkla
- `keywords`: arama; hüküm iddiası değil

## Sonraki diller (öneri)

1. **id** — Endonezya (LTR, büyük pazar)
2. **ms** — Malezya / Brunei (id'ye yakın)
3. **en** — diaspora + mağaza
4. **bn** — Bangladeş (ileride; ayrı editoryal iş)
5. **ur** — Pakistan (RTL; ayrı UI turu)

Urduca şimdilik listede yok.
